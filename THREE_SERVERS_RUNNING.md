### Option 2: Share Globally with ngrok (Recommended)

**One-time setup (3 minutes):**
```bash
# 1. Download: https://ngrok.com/download
# 2. Create account: https://ngrok.com/signup
# 3. Get auth token
# 4. Setup: ngrok config add-authtoken YOUR_TOKEN
```

**Then run these in 3 separate terminals:**

**Terminal 1:**
```bash
ngrok http 5000 --region us
# Copy the URL: https://xxxx-xxxx-xxxx.ngrok.io
```

**Terminal 2:**
```bash
ngrok http 8080 --region us
# Copy the URL: https://yyyy-yyyy-yyyy.ngrok.io
```

## 🔒 SECURITY NOTES

### When Sharing via ngrok:
- ✅ HTTPS secured
- ✅ Temporary URLs (session-based)
- ✅ Can revoke anytime
- ✅ Share only with intended users
- ✅ Don't post URLs publicly

### When Sharing on Local Network:
- ✅ Only on your WiFi
- ✅ Faster than ngrok
- ✅ No account needed
- ✅ Perfect for LAN testing

### Best Practices:
- ✅ Use strong passwords (shown in frontend)
- ✅ Don't use production data
- ✅ Monitor in ngrok dashboard
- ✅ Close tunnels when done
- ✅ Rotate auth tokens regularly
