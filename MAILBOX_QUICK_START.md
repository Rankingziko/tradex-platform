## 🔔 Notification Bell - How to Use

### Open Notifications
```
1. Click bell icon (🔔) in top-right corner
2. See all your notifications
3. Unread count shows as red badge
```

### Manage Notifications
- **Mark as Read:** Click "Mark as read" button on any notification
- **Mark All as Read:** Click "Mark all as read" at bottom
- **Delete:** Click trash icon to remove notification
- **Close:** Click outside or X button to close dropdown

### Notification Types
- 📈 **Market Alerts** - Price movements, market events
- 🔔 **Website Updates** - Announcements, maintenance
- 💱 **Trading Events** - Order fills, trade updates
- 💰 **Deposit Notifications** - Deposit confirmations
- 💸 **Withdrawal Status** - Withdrawal updates
- 🔄 **Transfer Updates** - Fund transfer status
- 📋 **Report Responses** - Admin replies to your reports
- ⚠️ **Alerts** - Important warnings

## 📬 Report Problem - How to Use

### Submit a Report
```
1. Click 📬 icon in top bar (or "Report a Problem")
2. Select issue category:
   - 🔧 Technical Issue
   - 👤 Account Problem
   - 💳 Payment Issue
   - 📈 Trading Problem
   - 🔐 Security Concern
   - ❓ Other
3. Choose priority level
4. Enter subject (title of issue)
5. Describe the problem in detail
6. Click "Submit Report"
```

### What Happens Next
- ✅ You see confirmation message
- ✅ Admin gets notified immediately
- ✅ Report appears in your Mailbox
- ✅ You'll get notifications on status updates
- ✅ Admin response appears in report details

## 📋 Mailbox Page - How to Use

### Access Mailbox
```
Method 1: Click "Mailbox" in sidebar (📬 icon)
Method 2: URL: http://localhost:3000/mailbox
```

### View Your Reports
- **See All Reports:** List shows all your submitted reports
- **Stats Dashboard:** Total, Open, In Progress, Resolved counts
- **Status Colors:**
  - 🟡 Yellow = Open
  - 🔵 Blue = In Progress
  - 🟢 Green = Resolved
  - ⚫ Gray = Closed

### Track Status
1. Click any report to see details
2. View:
   - Category & Priority
   - Full message
   - Current status
   - Submission date
   - Admin response (if available)
3. Right panel shows full details

### Report Info
```
Each report shows:
- Subject (title)
- Status badge
- Category emoji
- Priority level
- Submission date/time
- ✓ Replied badge (if admin responded)
```

## 🎯 Common Tasks

### Report a Technical Problem
```
1. Click 📬 Report Button
2. Select: Technical Issue
3. Priority: High (if urgent) or Medium (normal)
4. Subject: "Chart not loading"
5. Message: "When I go to /charts page, the chart is blank"
6. Click Submit
7. Check Mailbox for status updates
```

### Check Payment Status
```
1. Look for notification about payment
2. Click bell to see deposit/withdrawal notifications
3. Or go to Mailbox to find related report if issue
```

### Report Account Issue
```
1. Click 📬 Report Button
2. Select: Account Problem
3. Describe what's wrong with your account
4. Admin will review and respond
5. See response in Mailbox page
```

### Track All Issues
```
1. Go to Mailbox (sidebar)
2. See stats at top (total, open, in progress, resolved)
3. Click any report to see full details
4. See admin response on right side
```
